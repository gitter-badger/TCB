A simple, Customize your Login Page with your Logo and Your site theme

== Description ==
Customize Login Page is a simple plugin gives you the ability to make your Wordpress Custom Login Page with your Logo and Your site theme

== Installation ==

This section describes how to install the plugin and get it working.
1. Upload `customize-login-page` to the `/wp-content/plugins/` directory
1. Activate the plugin through the 'Plugins' menu in WordPress

== Frequently Asked Questions ==

= Where can I find the setting after activate the plugin =

Go to Settings -> Customize Login Page
